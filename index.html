Git Better

Let's practice doing an un-do commit in local working directoy!

Following command will remove the last commit and new changes will need to be staged and committed. reset will undo the history and rewrite it. best used for local working files.

git reset HEAD~

git reset will also remove from the stage backing in the working directory.

git revert will undo the most recent commit and create a new commit, the bad commit stays in the history.
